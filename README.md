# 📊 簡易・青色申告サポートシステム

個人事業主の確定申告（青色申告）をサポートするための、売上・経費管理システムです。
開発の生産性を最大化するため、**生成AI（Gemini）をペアプログラミングの相手として活用**し、フルスタックでの高速開発に挑戦しました。

## 🚀 主な機能
- **取引の登録（Create）**：売上や経費の日付、金額、勘定科目の登録
- **取引履歴の一覧表示（Read）**：データベースから取得した明細のリアルタイム表示
- **合計金額の自動計算**：登録・削除時に連動して売上・経費・利益を自動集計
- **取引の削除（Delete）**：REST API（DELETEメソッド）を用いたデータの動的削除

## 🛠️ 使用技術
- **バックエンド**：Java 21 / Spring Boot 4.0 / Spring Data JPA
- **フロントエンド**：HTML5 / JavaScript (Fetch APIによる非同期通信) / Tailwind CSS
- **データベース**：PostgreSQL 18
- **開発ツール**：VS Code / pgAdmin / GitHub

## 💡 AIの活用とこだわり
単なるコードのコピペではなく、エラーログの分析とプロンプトによる指示出しを主体的に行いました。
開発途中、AIがアーキテクチャ層（ControllerとService）の役割を混同したコードを提示した際には、システムの全体構造を意識していたため自身でエラーを指摘・軌道修正し、成果物を完成させました。
